## Приложение для тренеров спортивных клубов
Работая в продуктовой компании, которая специализируется на создани приложений для спортивых клубов, я поддреживал приложение для тренеров, и также разрабатывал новое приложение для администраторов спортивных клубов.
В поддержку приложения для тренеров входило:
- Правки багов
- Написание автотестов для функционала приложения
а именно:
- Полная проверка всего стека навигации на проекте
- Покрытие тестами новых больших разделов
- Покрытие тестами функционала уведомлений
- Покрытие тестами обработки ошибок в приложении
- Создание новых больших функций для приложения,
а именно:
- Раздел с достижениями клиента для отслеживания различных показателей
- Раздел с оформлением оплаты тренировок
- Функционал уведомлений
- Рефакторинг всего приложения под новые версии флаттера и пакетов
- Внедрил функционал для отладки различных ошибок для сервиса поддержки
- А также множество различных задач по внедрению нового функционала в существующие функции
